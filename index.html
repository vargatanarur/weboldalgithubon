Weboldal publikálása GitHub Pages segítségével

Egyszerű útmutató statikus weboldal közzétételéhez.

Mi az a GitHub Pages?

A GitHub Pages egy ingyenes szolgáltatás, amellyel statikus weboldalakat tudsz közzétenni közvetlenül egy GitHub repóból. Ideális portfólióhoz, projektek bemutatásához vagy egyszerű infóoldalakhoz.

Lépések a publikáláshoz

1. GitHub fiók és új repository

  • Jelentkezz be a github.com oldalra.
  • Kattints a New (új repo) gombra.
  • Adj nevet a repónak (pl. my-website vagy felhasznalonev.github.io).
  • Állítsd Public-ra, majd hozd létre.

2. Fájlok hozzáadása

Hozz létre helyben két fájlt: index.html és style.css.

  • Töltsd fel őket a repóba (Add file → Upload files), vagy használd a Git parancssort.
  • Ügyelj rá, hogy az index.html a repo gyökerében legyen.

3. GitHub Pages bekapcsolása

  • Nyisd meg a repó Settings menüjét.
  • Keresd meg a Pages részt.
  • A Source alatt válaszd ki a main (vagy master) ágat.
  • Mentés után kapsz egy URL‑t, ahol az oldalad elérhető lesz.

4. Az oldal elérése és frissítése

Néhány perc múlva az oldalad elérhető lesz a megadott URL‑en, például: https://felhasznalonev.github.io/ vagy https://felhasznalonev.github.io/repo-nev/.

Ha módosítod az index.html vagy style.css fájlt és újra feltöltöd/commitolod, a GitHub Pages automatikusan frissíti a weboldalt.

Tippek

  • Mindig ellenőrizd, hogy a CSS fájl neve és elérési útja megegyezik a HTML-ben megadottal.
  • Használj beszédes commit üzeneteket (pl. “Stílus frissítése”).
  • Ha nem látszik az oldal, várj pár percet, majd frissítsd az oldalt a böngészőben.
